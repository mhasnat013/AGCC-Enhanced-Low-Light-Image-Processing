# AGCC-Enhanced-Low-Light-Image-Processing
Research-based implementation and enhancement of the AGCC framework for low-light image enhancement. Includes reproduction of a 2024 research paper, evaluation using PSNR, SSIM, NIQE, and BRISQUE, and an improved hybrid model using CLAHE, Bilateral Filtering, Wavelet Denoising, Canny Edge Detection, and Morphological Operations.
